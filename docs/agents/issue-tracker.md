# Issue tracker: GitHub

Issues are tracked using GitHub Issues in this repository.

## Commands

- **List issues:** `gh issue list`
- **View issue:** `gh issue view <number>`
- **Create issue:** `gh issue create --title "<title>" --body "<body>"`
- **Add comment:** `gh issue comment <number> --body "<body>"`
- **Edit issue:** `gh issue edit <number> --add-label "<label>"`
- **Close issue:** `gh issue close <number>`

## Guidelines

- Always check if an issue already exists before creating a new one.
- Use labels to indicate the state of the issue.
- When working on an issue, refer to it by its number (e.g., "Fixes #123").
