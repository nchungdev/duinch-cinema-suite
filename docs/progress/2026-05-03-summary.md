## How to Run PROD

### Method 1: Docker Compose (Recommended)
This is the standard way to run all services (DB, Redis, API, UI, Crawler) in production mode.
1.  **Configure `.env`**: Ensure `TMDB_READ_ACCESS_TOKEN` and `HOST_STORAGE_PATH` are set.
2.  **Launch**:
    ```bash
    docker compose up -d --build
    ```
3.  **Access**:
    - Web UI: `http://localhost:8087`
    - API: `http://localhost:8086`

### Method 2: Manual Build (Local Testing)
1.  **Backend**:
    ```bash
    cd duinch-cinema/backend
    uvicorn app.main:app --host 0.0.0.0 --port 8086
    ```
2.  **Web**:
    ```bash
    cd duinch-cinema/web
    npm run build
    npx serve -s dist -p 8087
    ```
